# Weather Data Analysis and Visualization Using Streamlit

This project is a Python-based Streamlit web application for analyzing and visualizing weather data.

## Features
- Upload weather data CSV file
- Filter data by location and date
- Visualize temperature, humidity, precipitation, and wind speed
- Interactive and user-friendly dashboard

## Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib

## How to Run
1. Install requirements:
   pip install -r requirements.txt
2. Run the app:
   python -m streamlit run weather_dashboard.py
